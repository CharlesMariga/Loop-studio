# Loopstudios Website

Loopstudios landing page with Tailwindcss [Frontend Mentor Challenge](https://www.frontendmentor.io/challenges/loopstudios-landing-page-N88J5Onjw)

## Usage

Install dependencies

```
npm Install
```

Run Tailwind CLI

```
npm run watch
```

Build for production

```
npm run build
```
